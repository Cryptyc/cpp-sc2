Build instructions

Prerequisites

  1. Download and install CMake.
  2. A compiler with C++14 support.
  3. If you are already cloned the repository, make sure a recursive clone of the project is done to download all submodules. If you already cloned the repository without specifying --recursive you can initialize and checkout the required submodules with the following git command:
    1 $ git submodule update --init --recursive
  4. Windows: Download and install Visual Studio 2017
  5. Linux: Install 'gcc-c++'.
  6. Linux: Install the 'make' utility.
  7. OS X: Install XCode.
  8. OS X: Install XCode command-line tools.

## Windows (Visual Studio 2017)

1 :: Clone the project.
2 $ git clone --recursive https://github.com/alkurbatov/cpp-sc2.git
3 
4 :: Enter the working directory.
5 $ cd cpp-sc2
6 
7 :: Create build directory.
8 $ mkdir build
9 $ cd build
10 
11 :: Create Visual Studio project files.
12 $ cmake ../ -G "Visual Studio 15 2017 Win64"
13 
14 :: Build the project using Visual Studio.
15 $ start cpp-sc2.sln
